# 데이터 정제 작업
- 데이터 정제는 제공된 데이터의 수학 문제와 파이썬 코드를 직접 눈으로 비교해보면서 매칭이 안되는 문제가 있는 지 검수하였습니다.
- 문제의 숫자와 파이썬 코드의 숫자가 매칭이 안되는 경우 파이썬 코드 부분 수정 후 정답값도 수정하였습니다.
# 데이터 EDA
- 문제 유형별 특성을 파악하기 위해 문장 길이, 형태소별 워드 클라우드로 시각화 해보았습니다.<br>
